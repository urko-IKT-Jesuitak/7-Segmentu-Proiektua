* **Breadboard eta kableak** 🔌

## 🔌 Hardware Konexioa
Konexioak honela egin ditugu gure kodean:
* **a** -> Pin 6 | **b** -> Pin 5 | **c** -> Pin 4 | **d** -> Pin 3
* **e** -> Pin 2 | **f** -> Pin 7 | **g** -> Pin 8
* **Katodo Komuna** -> **GND**
